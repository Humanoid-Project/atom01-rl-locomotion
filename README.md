# atom01-rl-locomotion

Reinforcement learning training for the **atom01** humanoid robot using [![GitHub](https://img.shields.io/badge/Isaac%20Lab-181717?logo=github&logoColor=white)](https://github.com/isaac-sim/IsaacLab).

This project focuses on teaching the atom01 robot static balance and locomotion through RL, starting from stable standing and progressively moving toward walking.

## Robot

The atom01 robot model is based on the open-source project:
